# clustering clients
<h1>Clusterização de clientes usando uma dataset do kaggle, com os gastos de cartão de crédito, para demonstrar a aplicação de algumas técnicas de ML.</h1>



 <!-- Explicação do projeto -->
<h2 align="left"> 🧾 Descrição do Projeto</h2>
<p align="left"> Projetos feitos para estudo de Machine Learning, para demonstrar técnicas de PCA, normalização, encontrar o número ideal de clusters e utilização do metódo K-means para agrupamento de clientes.
</p>
